# About

SDL samples.

Install SDL2 with

```bash
sudo apt install libsdl2-dev libsdl2-mixer-dev
```

command.

Run

```bash
scons [target]
```

command to build samples (e.g. `scons hello` to build `hello.cpp` sample).
